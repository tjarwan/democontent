This overall schema also suggests several ways that different types of
conceptual change might occur.
1. The links between a CL and models change:
a. A new model is associated with a CL.
b. A model ceases to be associated with a CL.
2. There are adjustments to a model resulting in the constitutive features of
a CM within it being linked with a given CL.
3. There are changes to feature aggregators used for categorization that are
associated with a CL, CM, or model.
