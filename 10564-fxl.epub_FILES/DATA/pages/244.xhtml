Consider another case: wave and particle theories of light and two-slit
experiments. We are quite capable of imagining both sets of dispersion
patterns. But if we are thinking of light as made up of particles, one pat-
tern may seem intuitively obvious, because that is what is produced when
we run a mental simulation of the propagation of particles through the
slits. If we think of light as made up of waves and simulate the results, the
other pattern may seem intuitively obvious. If we are assuming the aptness
of one model naively—that is, unaware that there are other models—the
intuitiveness seems to have the force of necessity. Indeed, if the particle
model is apt for this problem, light must behave in this way. But with the
right sort of critical distance, we can see that the kind of “necessity” that is
really implied by intuitive judgments is not really objective necessity. First
and foremost, it is the kind of necessity involved with a rule: in this model,
this principle is treated as necessary. Second, there is an additional kind
of necessity imputed to the extent that this model is apt for a particular
sort of problem: if (or to the extent that) the model is apt, the world must
also behave thus. In classical optics, this generated trenchant paradoxes,
because in some situations light seems to behave like a particle (the particle
model is apt), and in others it seems to behave like a wave (the wave model
is apt). The paradox is partially (though only partially) resolved by remem-
bering that any given model might be apt for some cases but not for others.
A good part of scientific competence consists in knowing which models to
use in which situations, but puzzle cases like this one also drive us to seek
a model that can accommodate cases that thus far need to be approached
through different models.
