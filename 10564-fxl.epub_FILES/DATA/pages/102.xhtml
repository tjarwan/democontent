What these examples show is that there are a number of distinct ways
that a model such as a map can represent its subject matter, but each must
do so in some particular way. No projection is both Mercator and sinusoidal.
And what choices one makes will have consequences for what information
is preserved, excluded, or distorted, and consequently what information
can reliably be extracted from the model. The topological rail map excludes
all information about surface geometry and magnitude. The Mercator and
sinusoidal projections include such information, but in different forms.
One needs to have a set of techniques for reading information off a particu-
lar type of map that is appropriate to that map. Many mapping techniques,
when used for large-scale maps of small areas of the earthâ€™s surface, such as
metropolitan Boston, will produce maps in which distance information is
preserved well enough that one can calculate real-world distance by mea-
suring distance on the map. (Greater distances can be calculated in a more
complicated way if one knows what type of projection is used.) The same
technique will not work with a topological map.
